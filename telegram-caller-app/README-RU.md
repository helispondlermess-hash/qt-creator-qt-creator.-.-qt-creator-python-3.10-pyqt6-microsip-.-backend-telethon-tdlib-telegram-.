# Telegram Caller - десктопное приложение для звонков

## Описание

Полноценное десктопное приложение для голосовых звонков через Telegram с современным тёмным дизайном в стиле синего/фиолетового.

## Функционал

### Авторизация
- Вход по номеру телефона
- Поддержка 2FA (двухфакторная аутентификация)
- Автоматическое сохранение сессии

### Звонки
- Голосовые звонки на номера телефонов
- Звонки по @username
- Интерфейс набора номера (Dial Pad)
- Таймер длительности звонка

### Управление
- Mute микрофона
- Выбор устройства ввода (микрофон)
- Выбор устройства вывода (динамики)
- Заглушка для включения камеры (видеозвонок)

### Контакты
- Загрузка контактов из Telegram
- Поиск по контактам
- Быстрый звонок двойным кликом

## Требования

- Python 3.10+
- Windows 10/11 (для EXE)
- Микрофон и динамики

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Сборка в EXE (Windows)

```bash
pyinstaller specs/telegram_caller.spec
```

Результат: папка `dist/TelegramCaller/` с TelegramCaller.exe
